# Find Skill

This skill provides a wrapper for the `clawdhub` CLI to manage skills.

## Usage

```bash
# List skills
clawd run find-skill list

# Search for skills
clawd run find-skill search <query>

# Sync skills
clawd run find-skill sync
```
